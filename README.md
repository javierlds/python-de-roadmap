# Python Data Engineering Roadmap

Repositorio personal de aprendizaje para mi transición hacia **Data Engineer Senior** y, a futuro, **AI Engineer**. Sigue un roadmap estructurado de 16 fases (Python → Cloud → ML/IA).

## 👨‍💻 Sobre mí

Soy Javier Páez, con 15+ años de experiencia en **SQL y PL/SQL sobre Oracle**, base junior en Java y Spring Boot. Este repo documenta mi journey aprendiendo Python desde fundamentos hasta nivel ingeniero de datos profesional.

## 🎯 Objetivo

Construir, paso a paso, las habilidades necesarias para:
- Diseñar y operar pipelines de datos a escala
- Migrar sistemas Oracle on-premise a cloud (AWS / Databricks)
- Sentar bases sólidas para evolucionar a AI Engineer

## 📂 Estructura del repo

```
.
├── fase-1-python-base/      # Fundamentos de Python (con paralelos a PL/SQL)
```

> La estructura se irá llenando conforme avance en el roadmap.

## 🛠️ Stack

- **Lenguaje:** Python 3.13
- **Entornos:** `venv` + `pip`
- **Editor:** VS Code
- **Versionado:** Git + GitHub

## 🚀 Setup local

```bash
# Clonar el repo
git clone https://github.com/javierlds/[NOMBRE-DEL-REPO].git
cd [NOMBRE-DEL-REPO]

# Crear y activar entorno virtual
python -m venv .venv
.venv\Scripts\activate     # Windows
# source .venv/bin/activate  # Mac/Linux

# Instalar dependencias
pip install -r requirements.txt
```
